Informações Básicas

As chaves raiz definem o esqueleto do seu currículo. Certifique-se de preencher o title para garantir uma boa indexação por ATS.

{
  "name": "Seu Nome Completo",
  "title": "Seu Cargo (Ex: Software Engineer)",
  "objective": "Um breve resumo das suas ambições...",
  "additional_info": "Cursos extras, voluntariado ou hobbies."
}

Contatos e Links

Autolink Engine

O motor de renderização detecta automaticamente o formato dos dados para gerar links funcionais:

  • email: vira um link mailto
  • whatsapp/phone: link para chat direto
  • linkedin/github: URLs tratadas para abertura em nova aba
"contact": {
  "location": "São Paulo, SP",
  "whatsapp": "(11) 99999-9999",
  "email": "contato@dev.com",
  "linkedin": "linkedin.com/in/perfil"
}

Habilidades Técnicas

// Cada entrada é renderizada dinamicamente.
// O nome da chave vira o título do grupo. 

"technical_skills": {
  "linguagens": ["Go", "Python", "SQL"],
  "infraestrutura": ["AWS", "Docker", "K8s"]
}

Resultado Visual:

Linguagens: Go, Python, SQL

Infraestrutura: AWS, Docker, K8s

Projetos Pessoais

Ideal para destacar portfólio. O campo tech permite listar as stacks utilizadas em formato de tags.

"projects": [
  {
    "name": "API de Gestão",
    "github": "https://github.com/perfil/api",
    "tech": ["Node.js", "Redis"],
    "description": "Microsserviço de alta disponibilidade."
  }
]

Histórico Profissional

Organize por ordem cronológica inversa (do mais recente para o mais antigo).

"experience": [
  {
    "company": "Empresa S/A",
    "role": "Desenvolvedor Pleno",
    "period": "2021 - Presente",
    "description": "Responsável por X e Y..."
  }
]

Cover Letter

O sistema detecta a presença desta chave e insere automaticamente uma quebra de página para impressão.

Use \n\n no JSON para representar quebras de parágrafo no texto final.
